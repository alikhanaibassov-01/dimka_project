## Как перенести в Word

1. Откройте Word, создайте документ с полями по ГОСТ вашего вуза.
2. Скопируйте текст из каждого `.md` файла по порядку.
3. Заголовки «ВВЕДЕНИЕ», «1 АНАЛИЗ…» — стиль **Заголовок 1**.
4. Подпункты «1.1 …» — стиль **Заголовок 2**.
5. Вставьте **автособираемое оглавление** (Ссылки → Оглавление).
6. Диаграммы из главы 2 перерисуйте в Word (Вставка → SmartArt) или экспортируйте из Mermaid (mermaid.live).
7. Скриншоты интерфейса: `https://dimkaproject-production.up.railway.app`
