# ЗАКЛЮЧЕНИЕ

В ходе выполнения дипломной работы была достигнута поставленная цель: спроектирован и реализован веб-маркетплейс **QazMarket** для продажи продукции казахстанского производства.

**Основные результаты:**

1. Проведён анализ рынка e-commerce и маркetplace в Казахстане, обоснован выбор способов оплаты (Kaspi, оплата при получении) с учётом недоступности Stripe для merchant-аккаунтов в РК.

2. Спроектирована трёхуровневая архитектура «клиент — Express API — SQLite», ER-модель базы данных, роли пользователей и сценарии оформления заказа.

3. Реализован полнофункциональный веб-сервис:
   - каталог с фильтрацией и многоязычными карточками (kk / ru / en);
   - корзина и checkout с доставкой и самовывозом;
   - регистрация, личный кабинет с профилем (фамилия, имя, телефон);
   - административная панель (CRUD товаров, управление заказами);
   - загрузка и обработка изображений;
   - оплата Kaspi (перевод + QR) и COD;
   - карта точки самовывоза (пр. Аль-Фараби, 93) со ссылкой на 2GIS.

4. Приложение развёрнуто на Railway и доступно по постоянному URL.

5. Проведено функциональное тестирование (15 тест-кейсов), рассмотрены вопросы охраны труда и информационной безопасности (bcrypt, сессии, RBAC).

**Практическая значимость:** проект демонстрирует полный цикл разработки веб-приложения и может служить основой для коммерческого маркетплейса отечественной продукции.

**Направления развития:** интеграция Kaspi Merchant API, PostgreSQL, облачное хранилище, роль продавца, мобильное PWA-приложение.

Таким образом, все задачи дипломной работы выполнены, система QazMarket работоспособна и готова к демонстрации на защите.
