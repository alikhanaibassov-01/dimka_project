## АННОТАЦИЯ

Дипломная работа посвящена проектированию и реализации платформы объявлений формата Marketplace QazMarket, предназначенной для продажи товаров казахстанского производства. В работе проведён анализ существующих платформ электронной торговли, спроектированы клиент-серверная архитектура приложения и реляционная база данных SQLite. Реализованы каталог товаров с фильтрацией и трёхъязычным интерфейсом (казахский, русский, английский), корзина, оформление заказов с доставкой и самовывозом, регистрация и личный кабинет покупателя, административная панель управления товарами и заказами, а также оплата через Kaspi и при получении. Приложение разработано с использованием технологий Node.js, Express, SQLite, HTML, Tailwind CSS и JavaScript и развёрнуто на облачной платформе Railway. Проведено функциональное тестирование основных пользовательских сценариев, рассмотрены вопросы охраны труда и информационной безопасности.

## АҢДАТПА

Дипломдық жұмыс қазақстандық өндіріс тауарларын сатуға арналған Marketplace форматындағы QazMarket хабарландыру платформасын жобалау мен іске асыруға арналған. Жұмыста қолданыстағы электрондық сауда платформаларына талдау жүргізілді, қосымшаның клиент-серверлік архитектурасы мен SQLite реляциялық дерекқоры жобаланды. Сүзгілеуі бар және үш тілді интерфейсті (қазақ, орыс, ағылшын) тауарлар каталогы, себет, жеткізу және өзімен алып кету арқылы тапсырыс рәсімдеу, тіркелу мен сатып алушының жеке кабинеті, тауарлар мен тапсырыстарды басқарудың әкімшілік панелі, сондай-ақ Kaspi арқылы және алған кезде төлеу мүмкіндігі іске асырылды. Қосымша Node.js, Express, SQLite, HTML, Tailwind CSS және JavaScript технологияларын қолдану арқылы әзірленіп, Railway бұлттық платформасында орналастырылды. Негізгі пайдаланушы сценарийлеріне функционалдық тестілеу жүргізіліп, еңбек қорғау мен ақпараттық қауіпсіздік мәселелері қарастырылды.

## ABSTRACT

The diploma project is devoted to the design and implementation of QazMarket, a marketplace-style classifieds platform for selling goods produced in Kazakhstan. The work includes an analysis of existing e-commerce platforms, the design of a client-server application architecture and a relational SQLite database. The implemented features include a product catalog with filtering and a trilingual interface (Kazakh, Russian, English), a shopping cart, order placement with delivery and pickup options, customer registration and a personal account, an administrative panel for managing products and orders, as well as payment via Kaspi and cash on delivery. The application was developed using Node.js, Express, SQLite, HTML, Tailwind CSS and JavaScript, and deployed on the Railway cloud platform. Functional testing of the main user scenarios was carried out, and issues of occupational safety and information security were considered.
